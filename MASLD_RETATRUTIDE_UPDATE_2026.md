## 🧠 Mechanism: Why Triple Agonist Wins for MASLD

```
Retatrutide's 3 receptors:
┌─────────────────────────────────────────────────────────┐
│ 1. GLP-1    → Slow gastric emptying                      │
│             → Insulin release                            │
│             → Appetite suppression                       │
├─────────────────────────────────────────────────────────┤
│ 2. GIP      → 9x potency vs natural hormone              │
│             → Enhanced insulin sensitivity               │
│             → Fat burning (white → beige)               │
├─────────────────────────────────────────────────────────┤
│ 3. Glucagon → Liver fatty acid oxidation                 │
│             → ↑ Resting energy expenditure              │
│             → Even while sleeping!                      │
└─────────────────────────────────────────────────────────┘

Insulin from GLP-1 + GIP offsets glucagon's glucose-raising effect
= Fat burning WITHOUT blood sugar spike
```

## 📈 Clinical Data Summary

### Phase 2 (NCT04881785) - 338 adults, 48 weeks

| Endpoint | Result | Notes |
|----------|--------|-------|
| **Body weight** | -24.2% (12mg) | Still declining, no plateau |
| **Liver fat (MRI-PDFF)** | -86% | Unprecedented |
| **Normal liver fat** | 93% patients | MASLD resolution |
| **Prediabetes → Normal** | 72% | Strong glycemic effect |
| **Safety** | Well tolerated | GI side effects managed |

### Key Insight: The Curve is Still Falling

```
Typical obesity drug: plateau at 16-24 weeks
Retatrutide:         still losing at 48 weeks
Implication:         >30% weight loss possible at 72 weeks
```
