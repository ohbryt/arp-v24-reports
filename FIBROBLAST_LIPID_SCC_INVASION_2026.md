# Literature Alert: Fibroblast Lipid Cues in SCC Invasion
## Nature Cancer | DOI: 10.1038/s42255-026-01514-y

**Published:** April 2026  
**Title:** Tissue-specific fibroblast lipid cues impose the rate of epithelial cancer invasion  
**Journal:** Nature Cancer  

---

## Key Findings

### 1. Fibroblast Type Determines SCC Invasion Rate

```
┌─────────────────────────────────────────────────────────┐
│  Lung/Oral Fibroblasts (Lipid-rich) → HIGH invasion   │
│  Cutaneous Fibroblasts (Lipid-poor) → LOW invasion    │
└─────────────────────────────────────────────────────────┘

SCC subtypes tested: Cutaneous, Oral, Lung
Result: All 3 showed MORE invasion with lung/oral fibroblasts
```

### 2. Lipid Transfer Mechanism

```
Fibroblasts ──(Lipids)──→ SCC cells
                        ↓
               EMT activation
                        ↓
               INVASION increase
```

### 3. Tissue-Specific Lipid Signatures

| Fibroblast Source | Lipid Type | Target Cancer |
|-------------------|------------|---------------|
| **Oral** | Sphingolipids (SMs) | Oral SCC |
| **Lung** | Triglycerides (TGs) | Lung SCC |
| **Cutaneous** | Lipid-poor | Skin SCC (low invasion) |

### 4. Mechanistic Insights

```
Targeting approaches:
├── Fibroblast lipid synthesis → inhibits invasion
├── Lipid uptake blockade → inhibits invasion  
└── Lipid breakdown inhibition → inhibits invasion
```
