## 1. Citrulline-Gut Aging Connection

### From Science Advances 2025 Paper

```
Citrulline effects relevant to gut:
├── Anti-inflammatory (inflammaging mitigation)
├── mTOR inhibition (cellular homeostasis)
├── Macrophage metabolism modulation
└── DNA damage protection

Gut-specific implications:
├── Intestinal barrier function
├── Gut-associated lymphoid tissue (GALT)
├── Microbiome composition
└── Metabolic endotoxemia
```

### Citrulline Sources

```
ENDOGENOUS:
├── Produced in gut (glutamine → citrulline via enterocytes)
├── Kidney converts citrulline to arginine
└── Declines with age (confirmed in mice)

EXOGENOUS:
├── Watermelon (highest natural source)
├── Other melons, cucumbers
├── Supplements (L-citrulline)
└── Protein-rich foods
```

## 2. Gut Aging Mechanisms

### Key Pathways

```
AGING GUT PHENOTYPE:
├── Gut barrier dysfunction (leaky gut)
├── Chronic low-grade inflammation
├── Microbiome dysbiosis
├── Immune senescence (GALT)
└── mTOR hyperactivation

THERAPEUTIC TARGETS:
├── mTOR (rapamycin, citrulline)
├── Tight junction proteins (zonulin)
├── AMPK activation
├── SIRT1 pathway
└── Autophagy induction
```

### Biomarkers

| Marker | Change in Aging | Intervention |
|--------|-----------------|--------------|
| Citrulline | ↓ (declines) | Supplementation |
| Zonulin | ↑ (barrier leak) | Tight junction support |
| LPS | ↑ (endotoxemia) | Prebiotics, barrier repair |
| IL-6 | ↑ (inflammation) | Anti-inflammatory |
| mTOR activity | ↑ (hyperactivation) | Inhibition |

## 3. Research Strategy

### Hypothesis

```
Citrulline supplementation → mTOR inhibition in gut immune cells
                            → Reduced intestinal inflammation
                            → Improved gut barrier integrity
                            → Decreased systemic endotoxemia
                            → Mitigation of inflammaging
```

### Proposed Studies

#### Study 1: Citrulline + Gut Barrier
```
Model: Aged mice (18-20 months)
Treatment: L-citrulline in drinking water (1-2% w/v)
Duration: 8-12 weeks
Readouts:
├── Gut barrier function (FITC-dextran permeability)
├── Tight junction protein expression (occludin, ZO-1)
├── Systemic LPS levels
├── Intestinal inflammation (IL-6, TNF-α)
└── Microbiome composition (16S rRNA)
```

#### Study 2: Citrulline + Intestinal Macrophages
```
Model: Primary intestinal macrophages + aged mice
Treatment: L-citrulline (dose-response)
Readouts:
├── mTOR activity (p-S6K1)
├── HIF1α and glycolysis markers
├── M1/M2 polarization
├── Cytokine production
└── Phagocytic function
```

#### Study 3: Combination Therapy
```
Citrulline + [Target] combination:

Option A: Citrulline + DGAT1 inhibitor
├── Mechanism: mTOR inhibition + lipid metabolism
├── Rationale: Gut lipid metabolism × mTOR
└── Potential: Metabolic anti-aging

Option B: Citrulline + Prebiotic
├── Mechanism: Microbiome + metabolite
└── Rationale: Synbiotic approach

Option C: Citrulline + Rapamycin
├── Mechanism: Dual mTOR inhibition
└── Rationale: Potent anti-aging
```

├── Mouse: 1-2% w/v in drinking water (~1-2 g/kg/day)
├── Equivalent human: ~6-12 g/day
├── Duration: 8-12 weeks (chronic)
└── Safety: Excellent (endogenous AA)

HUMAN SUPPLEMENTATION:
