## 8. Framework Application to Novel Compounds

### Our De Novo Peptide Designs

When evaluating novel compounds from our pipeline:

```
Step 1: Mechanism clarity
├── Clear pathway? → +2 points
├── Plausible but complex? → +1 point
└── Theoretical? → 0 points

Step 2: Data quality
├── Large RCT (1000+) → +3 points
├── Phase 2 data → +2 points
├── Phase 1/animal → +1 point
└── Anecdotal → 0 points

Step 3: Safety characterization
├── Long-term RCT safety → +3 points
├── Short-term data → +2 points
├── Preclinical only → +1 point
└── Unknown → 0 points

Step 4: Bucket assignment
- 8-10: Bucket 1 (FDA approved)
- 6-8: Bucket 2-3 (strong mechanism)
- 4-6: Bucket 3-4 (promising but limited)
- <4: Monitor only
```
