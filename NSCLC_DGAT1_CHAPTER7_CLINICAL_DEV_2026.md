### 7.2 Phase I/II 설계

```
Phase I/II 임상 시험 설계:

Phase I: 안전성 및 용량 결정 (2027)

대상:
├── 진행성 NSCLC (DGAT1 높은 발현)
├── 표준 치료 실패 후
└── 18-30명

목적:
├── MTD (Maximum Tolerated Dose) 결정
├── 안전성 프로파일 확립
├── PK/PD 확인
└── 용량 최적화

Phase II: 효능 확인 (2028)

대상:
├── 1차 이상 치료 실패 후 NSCLC
├── DGAT1 높은 발현 + biomarker positive
└── 40-60명

디자인:
├── 단일-arm 또는 무작위 배정
├── Pradigastat + Pembrolizumab
└── biomarker 기반 환자 선별

1차 평가:
├── ORR (客观应答率)
├── PFS (병진행 자유 생존)
└── DOR (응답 지속 시간)

2차 평가:
├── OS (전체 생존)
├── 안전성
└── biomarker 반응
```

## 10. 위험 평가 및 대응

### 10.1 주요 위험

```
위험 평가 매트릭스:

| 위험 | 영향 | 가능성 | 대응책 |
|------|------|--------|--------|
| 임상적 독성 (GI) | 높음 | 중간 | inhaled formulation, 저용량 시작 |
| 제한된 단일 효능 | 중간 | 중간 | biomarker 선별, 조합 치료 |
| 내성 발생 | 중간 | 높음 | 순차적 치료, 다중 기전 |
| 경쟁약 대비 열위 | 중간 | 중간 | 차별화 전략, 조합 강조 |
| IND 실패 | 높음 | 낮음 | 충분한 전임상 데이터 |
| 파트너십 실패 | 중간 | 중간 | 다각화 전략, 자금 준비 |
| 규제 지연 | 중간 | 중간 | 조기 대화, expedited programs |

상위 3 위험:
1. 위장 독성 (계통성 DGAT1 억제)
   → PF14 inhaled 또는 저용량 병용

2. 단일 약제 효능 제한
   → biomarker 선별 + 조합 치료

3. 내성 발생
   → 다중 기전 조합, 치료 순서 최적화
```

### 10.2 위험 대응 계획

```
위험 대응 상세:

1. 위장 독성:
   ├── 모니터링: 위장 증상 일지
   ├── 관리: 점진적 용량 증가
   ├── 대안: PF14-siDGAT1 inhaled
   └── 연루: 영양사, 위장병 전문의

2. 제한된 단일 효능:
   ├── 조합 치료로 efficacy 향상
   ├── biomarker로 responders 선별
   ├── 순차적 치료로 resistance 극복
   └── immunotherapy 조합으로 개선

3. 내성 발생:
   ├── 조기 biomarker 모니터링
   ├── Resistance mechanism 연구
   ├── 순차적 치료 전략
   ├── 새로운 기전의 약물 개발

4. 경쟁 대비:
   ├── 차별화: 대사 표적, 합성 치사
   ├── biomarker-driven approach
   ├── 조합 치료로서의 가치
   └── 환자 중심 가치 창출

5. 규제 지연:
   ├── 조기 FDA와의 소통
   ├── Breakthrough designation 신청
   ├── expedited programs 활용
   └── 파트너의 규제 역량 활용
```
